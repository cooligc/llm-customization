IPL GPT-Neo Retrieval-Augmented Generation Demo

This simple application demonstrates how you can build a custom language model and retrieval system over a dataset of Indian Premier League cricket events.

Purpose

Users can ask questions about match details, players, or events and receive answers generated by a fine-tuned GPT-Neo model. Answers are informed by a ball-by-ball dataset stored in data/IPL.csv. By combining a vector-based retriever with a language model, the system provides retrieval-augmented generation (RAG).

How it works

  1. Preprocessing: parse the CSV and convert each delivery into a short text passage.
  2. Embedding & Indexing: embed passages using a sentence-transformer and store vectors in a FAISS index for fast similarity search.
  3. Fine-tuning: optionally train GPT-Neo on the passages to adapt its language to cricket terminology.
  4. Querying: when a question is asked, the retriever finds relevant passages, which are prepended to the prompt sent to GPT-Neo. The model then generates an answer conditioned on both the question and the retrieved context.
